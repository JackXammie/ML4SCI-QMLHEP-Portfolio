# Task 8: Vision Transformer & Quantum Vision Transformer (QViT)

This project implements a Vision Transformer (ViT) for MNIST digit classification and explores a conceptual extension to a Quantum Vision Transformer (QViT).

## Overview
- Implemented patch-based image encoding
- Built transformer encoder architecture with self-attention
- Achieved high classification accuracy on MNIST
- Designed a quantum extension using parameterized circuits and entanglement

## Results
- Baseline ViT Accuracy: 97.12%
- Enhanced ViT Accuracy: 97.56%

## Key Insight
Increasing model complexity led to minimal performance gain while significantly increasing training time, highlighting the importance of efficiency in model design.
